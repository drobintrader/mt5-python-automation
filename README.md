This repository contains a suite of Python scripts designed to bridge the gap between data science and live execution on the MetaTrader 5 platform.

Key Features:

Robust Connection: Managed initialization and error handling for MT5.

Market Data Extraction: Automated fetching of OHLC and Tick data for backtesting.

Order Management: Template functions for opening/closing positions with SL/TP.

Tech Stack: Python, MetaTrader5 Library, Pandas.
